## 画像作成時の注意点

### スタイル
- **ビジネス書的な信頼感**: 過度に装飾しない、シンプルで洗練されたデザイン
- **配色**:
  - P/L: ブルー系（#3B82F6）
  - B/S: グリーン系（#10B981）
  - C/F: オレンジ系（#F97316）
- **フォント**: 読みやすい日本語フォント（メイリオ、游ゴシックなど）
- **サイズ**: 横幅1200px程度、高さは内容に応じて

### 内容
- **P/Lサンプル**: 実際の損益計算書の形式で、架空の数値例を記載
- **B/Sサンプル**: 左右対称のレイアウトで、資産の部・負債の部・純資産の部を明確に
- **C/Fサンプル**: 営業・投資・財務の3区分を色分けして表示
- **関連図**: P/L→B/S←C/Fの繋がりを矢印で視覚的に表現
- **フロー図**: 「日々の記録→月末集計→財務諸表作成→分析→改善」の流れを図示
