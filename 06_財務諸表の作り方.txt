6.1 財務諸表とは何か（復習）
--------------------
【内容】
3つの財務諸表の役割を再確認

【損益計算書（P/L）】
- いつ：一定期間（月次・年次）
- 何を：どれだけ儲かったか
- 用途：利益の把握、経営判断

【貸借対照表（B/S）】
- いつ：ある時点（月末・年度末）
- 何を：何を持っているか
- 用途：財務健全性の確認

【キャッシュフロー計算書（C/F）】
- いつ：一定期間（月次・年次）
- 何を：現金の出入り
- 用途：資金繰りの確認

【3つの関係性】
- P/Lの利益がB/Sの純資産を増やす
- P/Lの利益≠C/Fの現金増減
- 3つを見て初めて全体像が分かる

【HTML化のポイント】
- 3つの財務諸表の関係図
- 各財務諸表のサンプル画像

6.2 損益計算書（P/L）の作り方
--------------------
【内容】
P/Lを実際に作成する手順

【基本構造（おさらい）】
```
売上高                    1,000万円
- 売上原価                  600万円
--------------------------------
= 売上総利益（粗利）         400万円
- 販売費及び一般管理費       300万円
--------------------------------
= 営業利益                  100万円
+ 営業外収益                  5万円
- 営業外費用                 10万円
--------------------------------
= 経常利益                   95万円
+/- 特別損益                  0万円
--------------------------------
= 税引前当期純利益           95万円
- 法人税等                   25万円
--------------------------------
= 当期純利益                 70万円
```

【ステップ1：売上高の集計】
- レジ・売上台帳から集計
- 月ごとに記録
- 返品・値引きは差し引く

記入例：
  - 商品A売上：300万円
  - 商品B売上：500万円
  - サービス売上：200万円
  → 合計：1,000万円

【ステップ2：売上原価の計算】
売上原価 = 期首在庫 + 当期仕入 - 期末在庫

例：
  - 期首在庫：50万円
  - 当期仕入：600万円
  - 期末在庫：50万円
  → 売上原価：600万円

【ステップ3：売上総利益の計算】
売上総利益 = 売上高 - 売上原価
  = 1,000万円 - 600万円 = 400万円

粗利率 = 400万円 ÷ 1,000万円 = 40%

【ステップ4：販管費の集計】
- 人件費（給料、社会保険料）
- 家賃
- 広告費
- 通信費
- 消耗品費
- 水道光熱費
- 減価償却費
など

例：
  - 人件費：150万円
  - 家賃：50万円
  - 広告費：30万円
  - その他：70万円
  → 合計：300万円

【ステップ5：営業利益の計算】
営業利益 = 売上総利益 - 販管費
  = 400万円 - 300万円 = 100万円

【ステップ6：営業外損益の計算】
営業外収益：
  - 受取利息：1万円
  - 雑収入：4万円
  → 合計：5万円

営業外費用：
  - 支払利息（融資の利息）：10万円
  → 合計：10万円

【ステップ7：経常利益の計算】
経常利益 = 営業利益 + 営業外収益 - 営業外費用
  = 100万円 + 5万円 - 10万円 = 95万円

【ステップ8：税金の計算】
法人税等 = 税引前利益 × 税率
  = 95万円 × 約26% ≒ 25万円

※実際の税率は所得により異なる

【ステップ9：当期純利益の計算】
当期純利益 = 税引前利益 - 法人税等
  = 95万円 - 25万円 = 70万円

【HTML化のポイント】
- 各ステップを折りたたみ式で表示
- 記入用テンプレート（Excel/PDF）
- 自動計算ツール


6.3 貸借対照表（B/S）の作り方
--------------------
【内容】
B/Sを実際に作成する手順

【基本構造（おさらい）】
```
【資産の部】              【負債の部】
流動資産                 流動負債
  現金預金    200万円       買掛金      50万円
  売掛金     100万円       未払金      30万円
  在庫        50万円       短期借入金  100万円
  小計       350万円       小計       180万円

固定資産                 固定負債
  建物       200万円       長期借入金  300万円
  機械装置    100万円       小計       300万円
  小計       300万円
                         負債合計     480万円

                         【純資産の部】
                           資本金      100万円
                           利益剰余金   70万円
                           小計       170万円

資産合計     650万円      負債・純資産合計 650万円
```

【ステップ1：流動資産の集計】
現金預金：
  - 現金（レジ・手元）：10万円
  - 普通預金（A銀行）：100万円
  - 普通預金（B銀行）：90万円
  → 合計：200万円

売掛金：
  - A社：50万円（請求済・未入金）
  - B社：30万円
  - C社：20万円
  → 合計：100万円

在庫（商品）：
  - 商品A：20個 × 1万円 = 20万円
  - 商品B：10個 × 3万円 = 30万円
  → 合計：50万円

【ステップ2：固定資産の集計】
建物：
  - 取得価額：300万円
  - 減価償却累計額：-100万円
  → 簿価：200万円

機械装置：
  - 取得価額：150万円
  - 減価償却累計額：-50万円
  → 簿価：100万円

【ステップ3：流動負債の集計】
買掛金：
  - D社への仕入代金（未払）：50万円

未払金：
  - 広告費未払：20万円
  - その他：10万円
  → 合計：30万円

短期借入金：
  - 1年以内返済予定の融資：100万円

【ステップ4：固定負債の集計】
長期借入金：
  - 1年超の融資残高：300万円

【ステップ5：純資産の計算】
資本金：
  - 創業時の出資額：100万円

利益剰余金：
  - 過去の利益の累積：0万円
  - 今期の当期純利益：+70万円
  → 合計：70万円

【ステップ6：バランスチェック】
資産合計 = 負債合計 + 純資産合計
650万円 = 480万円 + 170万円 ✓

※必ず一致する（一致しない場合は計算ミス）

【HTML化のポイント】
- 左右対称のレイアウト
- 天秤のイラストでバランスを表現
- 記入用テンプレート
- バランスチェック機能


6.4 キャッシュフロー計算書（C/F）の作り方
--------------------
【内容】
C/Fを実際に作成する手順

【基本構造】
```
【営業活動によるキャッシュフロー】
  税引前当期純利益            95万円
  減価償却費                 30万円
  受取利息・支払利息         +5万円
  売上債権の増減             -20万円
  棚卸資産の増減             +10万円
  仕入債務の増減             +10万円
  小計                      130万円
  利息の受取額               +1万円
  利息の支払額               -10万円
  法人税等の支払額           -25万円
  営業活動CF                96万円

【投資活動によるキャッシュフロー】
  固定資産の取得             -50万円
  投資活動CF                -50万円

【財務活動によるキャッシュフロー】
  長期借入れによる収入       +200万円
  長期借入金の返済           -100万円
  財務活動CF                +100万円

現金及び現金同等物の増減     146万円
現金及び現金同等物の期首残高  54万円
現金及び現金同等物の期末残高 200万円
```

【ステップ1：営業CFの計算（簡易法）】
営業CF = 当期純利益 + 減価償却費 +/- 運転資金の増減

例：
  - 当期純利益：70万円
  - 減価償却費：30万円
  - 売掛金増加：-20万円（現金減）
  - 在庫減少：+10万円（現金増）
  - 買掛金増加：+10万円（現金増）
  → 営業CF：100万円

【ステップ2：投資CFの計算】
- 設備購入：-50万円（現金減）
- 設備売却：0万円
  → 投資CF：-50万円

【ステップ3：財務CFの計算】
- 融資受取：+200万円（現金増）
- 融資返済：-100万円（現金減）
- 配当金支払：0万円
  → 財務CF：+100万円

【ステップ4：現金増減の計算】
現金増減 = 営業CF + 投資CF + 財務CF
  = 100万円 - 50万円 + 100万円 = 150万円

【ステップ5：期末現金残高の確認】
期末残高 = 期首残高 + 現金増減
  = 50万円 + 150万円 = 200万円

→ B/Sの現金預金と一致するか確認

【小規模事業者向け簡易版】
難しければ、以下だけでもOK：

項目                    金額
--------------------------
期首現金残高            50万円
+ 営業による入金      1,200万円
- 営業による支出     -1,100万円
- 設備投資             -50万円
+ 融資受取            +200万円
- 融資返済            -100万円
--------------------------
期末現金残高           200万円

【HTML化のポイント】
- 3つの区分を色分け
- 簡易版と詳細版の両方を提供
- 記入用テンプレート
- P/L・B/Sとの整合性チェック


6.5 3つの財務諸表の関連性
--------------------
【内容】
P/L、B/S、C/Fがどう繋がるか

【関連図】
```
損益計算書（P/L）
  ↓ 当期純利益（70万円）
貸借対照表（B/S）
  純資産が70万円増加
  ↓
キャッシュフロー計算書（C/F）
  営業CFに含まれる
```

【具体例で確認】
前期末B/S：
  - 現金：50万円
  - 純資産：100万円

当期P/L：
  - 当期純利益：70万円

当期C/F：
  - 現金増減：+150万円

当期末B/S：
  - 現金：200万円（50万円 + 150万円）
  - 純資産：170万円（100万円 + 70万円）

【整合性チェック】
✓ P/Lの利益がB/Sの純資産に反映されているか
✓ C/Fの現金増減がB/Sの現金に反映されているか
✓ 3つの数字に矛盾がないか

- 3つの財務諸表を並べて表示
- 数字の繋がりを矢印で表現
- インタラクティブな関連図


6.6 会計ソフトを使った財務諸表作成
--------------------
【内容】
実際の会計ソフトでの作成方法

【主要会計ソフトの機能】
freee、マネーフォワード、弥生会計など

できること：
- 銀行・クレカ連携で自動仕訳
- レシート撮影で経費入力
- ワンクリックで財務諸表作成
- 確定申告書も自動作成

【基本的な流れ】
1. 銀行口座を連携
2. 取引が自動で取り込まれる
3. 勘定科目を確認・修正
4. 月次で確定
5. 財務諸表が自動生成

【手入力が必要な項目】
- 現金取引（レジ売上など）
- レシートのない経費
- 在庫の棚卸
- 減価償却費（初年度のみ設定）

【会計ソフトのメリット】
✓ 自動化で時間短縮
✓ 計算ミスが減る
✓ リアルタイムで財務状況把握
✓ 税理士との連携が楽

【会計ソフトの選び方】
- 操作性（初心者向けか）
- 料金（月額1,000-3,000円程度）
- サポート体制
- 銀行連携の対応数

【HTML化のポイント】
- 各ソフトの画面キャプチャ
- 比較表
- 外部リンク（各社公式サイト）
- 無料トライアルの案内


6.7 月次決算の進め方
--------------------
【内容】
毎月の財務諸表作成の実務

【月次決算とは】
- 毎月末に財務諸表を作成
- 月ごとの業績を確認
- 早期に問題を発見

【月次決算のスケジュール】
月初1-5日：
  - 前月の取引を確定
  - 会計ソフトに入力

月初6-10日：
  - 在庫棚卸
  - 未払金・売掛金の確認
  - 財務諸表作成

月初11-15日：
  - 前月の分析
  - 予算との比較
  - 改善策の検討

【月次決算でやること】
1. 現金実査（レジの現金を数える）
2. 銀行残高の確認
3. 売掛金・買掛金の確認
4. 在庫の棚卸
5. 未払経費の計上
6. 減価償却費の計上

【月次レポートの作成】
- P/L（月次・累計）
- B/S（月末時点）
- C/F（月次）
- 予算vs実績比較
- 前年同月比較

【HTML化のポイント】
- 月次決算スケジュール表
- チェックリスト
- レポートテンプレート


6.8 財務諸表の読み方・分析
--------------------
【内容】
作った財務諸表をどう活用するか

【P/Lの分析ポイント】
粗利率：
  - 売上総利益 ÷ 売上高
  - 目安：業種により異なる（30-60%）

営業利益率：
  - 営業利益 ÷ 売上高
  - 目安：5-10%以上

前月比・前年同月比：
  - 売上は伸びているか
  - 経費は増えすぎていないか

【B/Sの分析ポイント】
自己資本比率：
  - 純資産 ÷ 総資産
  - 目安：30%以上

流動比率：
  - 流動資産 ÷ 流動負債
  - 目安：100%以上（理想は150%以上）

現金比率：
  - 現金 ÷ 総資産
  - 目安：10-20%

【C/Fの分析ポイント】
営業CFがプラスか：
  - プラス：本業で稼げている
  - マイナス：本業で現金が減っている

フリーキャッシュフロー：
  - 営業CF + 投資CF
  - プラス：自由に使える現金がある

6.9 税務申告との関係
--------------------
【内容】
財務諸表と確定申告・法人税申告の関係

【個人事業主の場合】
確定申告に必要な書類：
  - 損益計算書（青色申告決算書）
  - 貸借対照表（青色申告決算書）
  - 確定申告書B

青色申告の特典：
  - 65万円控除（電子申告なら）
  - 赤字の繰越（3年間）
  - 家族への給料を経費にできる

【法人の場合】
法人税申告に必要な書類：
  - 損益計算書
  - 貸借対照表
  - 株主資本等変動計算書
  - 勘定科目内訳書
  - 法人税申告書

決算月から2ヶ月以内に申告

【税理士に頼むべきか】
自分でできる場合：
  - 個人事業主
  - 取引が少ない
  - 会計ソフトを使いこなせる

税理士に頼むべき場合：
  - 法人
  - 取引が多い
  - 節税したい
  - 時間がない

【HTML化のポイント】
- 個人と法人の違い一覧
- 税理士費用の目安
- 申告スケジュール表
