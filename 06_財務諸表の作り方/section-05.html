6.5 3つの財務諸表の関連性

P/L、B/S、C/Fは、それぞれ独立した表ではありません。 3つは密接に関連しており、セットで見ることで会社の全体像が理解できます。

このセクションでは、具体的な数値例を使って、3つの財務諸表がどのように繋がっているのかを確認していきましょう。

3つの財務諸表の関連図

主な関連性

1. P/Lの利益 → B/Sの純資産

P/Lで計算された当期純利益は、B/Sの純資産（利益剰余金）に加算されます。

例：当期純利益70万円 → B/Sの利益剰余金が70万円増加

2. P/Lの利益 ≠ C/Fの現金増減

P/Lで利益が出ていても、C/Fで現金が減っていることがあります。

理由：売掛金の増加、設備投資、融資返済などにより現金が減るため

3. C/Fの現金増減 → B/Sの現金残高

C/Fで計算された現金増減は、B/Sの現金預金の増減に反映されます。

C/Fの期末残高 = B/Sの現金預金（必ず一致）

具体例で確認

実際の数値例を使って、3つの財務諸表の繋がりを確認してみましょう。

【前期末のB/S】

資産の部

現金預金：50万円

その他資産：550万円

資産合計：600万円

負債・純資産の部

負債合計：500万円

純資産合計：100万円

合計：600万円

【当期のP/L】

売上高：1,000万円

売上原価：600万円

販管費：300万円

営業利益：100万円

経常利益：95万円

税金：25万円

→ 当期純利益：70万円

【当期のC/F】

営業CF：100万円

投資CF：-50万円

財務CF：+100万円

→ 現金増減：+150万円

【当期末のB/S】

資産の部

現金預金：200万円（50万円 + 150万円）

その他資産：450万円

資産合計：650万円

負債・純資産の部

負債合計：480万円

純資産合計：170万円（100万円 + 70万円）

合計：650万円

確認ポイント

  • P/Lの当期純利益70万円 → B/Sの純資産が100万円から170万円に増加
  • C/Fの現金増減+150万円 → B/Sの現金預金が50万円から200万円に増加
  • P/Lの利益70万円 ≠ C/Fの現金増減150万円（設備投資や融資の影響）

整合性チェックのポイント

3つの財務諸表を作成したら、以下のポイントで整合性をチェックしましょう。

✓ チェック1：P/Lの利益がB/Sの純資産に反映されているか

前期末純資産 + 当期純利益 = 当期末純資産

例：100万円 + 70万円 = 170万円 ✓

✓ チェック2：C/Fの現金増減がB/Sの現金に反映されているか

前期末現金 + C/Fの現金増減 = 当期末現金

例：50万円 + 150万円 = 200万円 ✓

✓ チェック3：3つの数字に矛盾がないか

P/L、B/S、C/Fの数字が辻褄が合っているか確認します。 例えば、P/Lで大きな利益が出ているのに、B/Sの純資産が減っている場合は、どこかに間違いがあります。

このセクションのまとめ

3つの財務諸表の関連性について、理解が深まったでしょうか？ P/L、B/S、C/Fは互いに繋がっており、セットで見ることが重要です。

✓ P/L → B/S

当期純利益が純資産を増やします。

✓ C/F → B/S

現金増減が現金預金を増減させます。

✓ P/L ≠ C/F

利益と現金増減は必ずしも一致しません。

✓ 整合性チェック

3つの数字に矛盾がないか確認しましょう。
