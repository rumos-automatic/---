6.3 貸借対照表（B/S）の作り方

貸借対照表（B/S）は、ある時点における会社の財産状況を示す表です。 左側に「資産の部」、右側に「負債の部」と「純資産の部」を記載します。

B/Sの特徴は、左側と右側が必ずバランス（釣り合い）することです。 このセクションでは、6つのステップでB/Sの作り方を学んでいきましょう。

重要なポイント

資産合計 = 負債合計 + 純資産合計

この式が必ず成り立ちます。バランスが取れていない場合は、どこかに計算ミスがあります。

B/S作成の6つのステップ

ステップ1：流動資産の集計

1年以内に現金化できる資産を集計します。

主な項目：

  • 現金預金：現金（レジ・手元）+ 普通預金（各銀行） = 200万円
  • 売掛金：請求済で未入金の代金 = 100万円
  • 在庫（商品）：棚卸により確認 = 50万円
  • → 流動資産合計：350万円

ステップ2：固定資産の集計

1年以上使用する資産を集計します。減価償却後の簿価を記載します。

例：

  • 建物：取得価額300万円 - 減価償却累計額100万円 = 簿価200万円
  • 機械装置：取得価額150万円 - 減価償却累計額50万円 = 簿価100万円
  • → 固定資産合計：300万円

ステップ3：流動負債の集計

1年以内に返済すべき負債を集計します。

主な項目：

  • 買掛金：仕入代金の未払い = 50万円
  • 未払金：広告費などの未払い = 30万円
  • 短期借入金：1年以内返済予定の融資 = 100万円
  • → 流動負債合計：180万円

ステップ4：固定負債の集計

1年超の返済期限がある負債を集計します。

例：

  • 長期借入金：1年超の融資残高 = 300万円
  • → 固定負債合計：300万円

ステップ5：純資産の計算

純資産は、会社の正味の財産です。

主な項目：

  • 資本金：創業時の出資額 = 100万円
  • 利益剰余金：過去の利益の累積 + 今期の当期純利益 = 70万円
  • → 純資産合計：170万円

ステップ6：バランスチェック

最後に、必ず左右のバランスを確認します。

確認：

資産合計 = 負債合計 + 純資産合計

650万円 = 480万円 + 170万円 ✓

注意：一致しない場合は、どこかに計算ミスがあります。各項目を再確認してください。
