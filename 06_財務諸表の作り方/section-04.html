6.4 キャッシュフロー計算書（C/F）の作り方

キャッシュフロー計算書（C/F）は、一定期間の現金の動きを示す表です。 P/Lで利益が出ていても、現金が減っていることがあります。その理由を明らかにするのがC/Fです。

このセクションでは、営業活動、投資活動、財務活動の3つに分けて、C/Fの作り方を学んでいきましょう。 小規模事業者向けの簡易版も紹介します。

基本構造

キャッシュフロー計算書の構造

【営業活動によるキャッシュフロー】

  小計                      130万円
  利息の受取額               +1万円
  利息の支払額               -10万円
  法人税等の支払額           -25万円
  営業活動CF                 96万円

【投資活動によるキャッシュフロー】
  固定資産の取得             -50万円
  投資活動CF                 -50万円

【財務活動によるキャッシュフロー】
  長期借入れによる収入       +200万円
  長期借入金の返済           -100万円
  財務活動CF                +100万円

現金及び現金同等物の増減     146万円
現金及び現金同等物の期首残高  54万円
現金及び現金同等物の期末残高 200万円

3つの区分

  • 営業CF：本業で稼いだ現金
  • 投資CF：設備投資などに使った現金
  • 財務CF：融資や返済による現金の増減

C/F作成の5つのステップ

ステップ1：営業CFの計算（簡易法）

営業CFは、本業でどれだけ現金を稼いだかを示します。 簡易法では、当期純利益に調整を加えて計算します。

営業CF = 当期純利益 + 減価償却費 +/- 運転資金の増減

例：

  • 当期純利益：70万円
  • 減価償却費：30万円（現金は出ていない）
  • 売掛金増加：-20万円（現金が減る）
  • 在庫減少：+10万円（現金が増える）
  • 買掛金増加：+10万円（現金が増える）
  • → 営業CF：100万円

ステップ2：投資CFの計算

投資CFは、設備投資などに使った現金を示します。

例：

  • 設備購入：-50万円（現金減）
  • 設備売却：0万円
  • → 投資CF：-50万円

ステップ3：財務CFの計算

財務CFは、融資の受取や返済による現金の増減を示します。

例：

  • 融資受取：+200万円（現金増）
  • 融資返済：-100万円（現金減）
  • 配当金支払：0万円
  • → 財務CF：+100万円

ステップ4：現金増減の計算

3つのCFを合計して、現金の増減を計算します。

現金増減 = 営業CF + 投資CF + 財務CF

= 100万円 - 50万円 + 100万円 = 150万円

ステップ5：期末現金残高の確認

期首残高に現金増減を加えて、期末残高を計算します。

期末残高 = 期首残高 + 現金増減

= 50万円 + 150万円 = 200万円

確認：この200万円は、B/Sの現金預金と一致するはずです。

小規模事業者向け簡易版

詳細なC/Fが難しい場合は、以下の簡易版でも十分です。 現金の動きを大まかに把握できます。

簡易版C/F

項目                    金額
--------------------------
期首現金残高            50万円
+ 営業による入金      1,200万円
- 営業による支出     -1,100万円
- 設備投資             -50万円
+ 融資受取            +200万円
- 融資返済            -100万円
--------------------------
期末現金残高           200万円

簡易版のポイント

入金と支出を大まかに分類するだけでOKです。 細かい調整は不要で、実際の現金の動きを把握することが目的です。
