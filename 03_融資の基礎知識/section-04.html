3.4 融資を受けるための条件

融資を受けるには、金融機関の審査を通過する必要があります。 審査では、事業の妥当性や返済能力など、様々な観点から判断されます。

このセクションでは、金融機関がどのような基準で融資を判断するのか、 そして必要な書類は何かを詳しく見ていきましょう。

審査で見られる5つのポイント

金融機関は、以下の5つのポイントを中心に融資の可否を判断します。 それぞれのポイントを理解して、しっかりと準備しましょう。

1 事業計画の妥当性

どんな事業をするのか、市場性はあるか、実現可能かどうかを判断します。

チェックされる内容

  • ✓ 事業の内容が明確か
  • ✓ 市場ニーズがあるか
  • ✓ 競合との差別化ができているか
  • ✓ 売上計画が現実的か
  • ✓ 事業の将来性があるか

2 返済能力

毎月の利益から返済できるか、キャッシュフローは健全かを判断します。 これが最も重要なポイントです。

チェックされる内容

  • ✓ 月の利益 > 月の返済額になっているか
  • ✓ キャッシュフローが黒字か
  • ✓ 売上計画が保守的か（楽観的すぎないか）
  • ✓ 経費計画が現実的か

重要： 「月の利益 > 月の返済額」が絶対条件です。 これが満たされないと、融資は受けられません。

3 自己資金

自分で用意できる資金がどれくらいあるかを確認します。 創業時は総投資額の1/3以上が目安です。

自己資金の目安

• 創業時： 総投資額の1/3以上

• 自己資金が多いほど有利

• 実例： 総投資額900万円なら、自己資金300万円以上

4 経営者の信用

経営者の信用情報や経験を確認します。

チェックされる内容

  • ✓ 過去の信用情報（ブラックリストに載っていないか）
  • ✓ 経営経験、業界経験があるか
  • ✓ 経営者としての資質があるか
  • ✓ 税金や公共料金の滞納がないか

5 担保・保証人

担保や保証人があれば審査に有利ですが、必須ではありません。

  • ✓ 担保（不動産など）があれば融資額が増える
  • ✓ 保証人がいれば審査に有利
  • ✓ 無担保・無保証人でも借りられる制度もある

必要書類（一般的）

融資申込みには、様々な書類が必要になります。 金融機関や融資制度によって異なりますが、一般的に必要な書類をまとめました。

書類名 内容
事業計画書 事業内容、売上計画、経費計画、返済計画などを記載
資金繰り表 毎月の現金の出入りを予測した表
損益計算書・貸借対照表 既存事業の場合のみ。過去の実績を示す
確定申告書 過去2〜3期分（既存事業の場合）
見積書 設備投資の場合、購入予定の設備の見積書
身分証明書・印鑑証明書 本人確認書類、印鑑証明書など

💡 書類作成のコツ

  • • 事業計画書は具体的に： 数値の根拠を明確にする
  • • 楽観的すぎない： 保守的な計画の方が信頼される
  • • 専門家に相談： 税理士や商工会議所に相談するのも有効
  • • 早めに準備： 書類作成には時間がかかります

このセクションのまとめ

融資を受けるための条件について、理解が深まったでしょうか？ 審査基準を理解して、しっかりと準備すれば、融資を受けられる可能性が高まります。

✓ 審査の重要ポイント

  • • 事業計画の妥当性
  • • 返済能力（最重要）
  • • 自己資金（1/3以上）
  • • 経営者の信用
  • • 担保・保証人

✓ 必要な準備

  • • 事業計画書の作成
  • • 資金繰り表の作成
  • • 必要書類の準備
  • • 専門家への相談
